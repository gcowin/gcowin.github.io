Visual Studio Code Notes
------------------------

Keyboard Shortcuts
------------------
CTRL+K,CTRL+S: keyboard shortcuts

CTRL+\ side-by-side editing
CTRL+1,CTRL+2,...: switch between editros

CTRL+SHIFT+E: move to explorer window

ALT+LEFT: navigate left - navigate editors
ALT+RIGHT: navigate right - navigating editors
ALT: enables fast scrolling
CTRL/CMD: multi-cursor editing
ALT+Click+Mouse: multi-cursor selecting
CTRL+SHIFT+V: markdown preview
CTRL+K,V: markdown side-by-side
CTRL+SPACE: intellisense
CTRL+SHIFT+P: command pallete 
CTRL+click: go to definition
F2: rename symbol 
CTRL+L: select current line
CTRL+P: quickly open file

Shift + Alt + F: format code

CTRL+K, W: close editor group
CTRL+K, CTRL+W: close all editor groups

Favorites
Ctrl+Shift+P Show All Commands workbench.action.showCommands
Ctrl+T Show All Symbols workbench.action.showAllSymbols

Navigating thru code history
Alt+->
Alt+<-

Ctrl+Shift+D: duplicate line
Alt+arrow-up: move line up

Default
{ "key": "ctrl+left", "command": "workbench.action.navigateBack" },
{ "key": "ctrl+right", "command": "workbench.action.navigateForward" }
