GTA V Cheats

Slow-motion mode
While playing the game, quickly press Y, Left, Right(2), X, RT, RB. The code can be entered up to four times to increase its effect. Enter it a fifth time to disable the code.

Change weather
While playing the game, quickly press RT, A, LB(2), LT(3), X. Each time the code is entered, it rotates through thunder, clear, and snowy weather.
